## Confidence Scoring Guide

- **0-20%**: Initial observations (1-5 writing sessions)
- **21-40%**: Emerging pattern (6-15 sessions)
- **41-60%**: Developing pattern (16-30 sessions)
- **61-80%**: Established pattern (31-50 sessions)
- **81-100%**: Strong pattern (51+ sessions)

**Special Cases:**
- Author explicit statement → Immediate 80%+ confidence
- Consistent behavior over 10+ sessions → 70%+ confidence
- Author correction → Update with high confidence
- Contradictory behavior → Note as "varies by project"

## Update Guidelines

### What to Extract
✅ **DO Extract:**
- Consistent stylistic choices
- Repeated structural patterns
- Explicit preferences stated by author
- Genre and content preferences
- Process preferences (planning, drafting, revising)
- Feedback response patterns

❌ **DON'T Extract:**
- One-time experiments
- Project-specific choices
- Temporary approaches
- Uncertain observations

### When to Update
- **During session**: Note patterns in session_notes.md
- **Every 10 interactions**: Review and extract clear patterns
- **Session end**: Update with observed patterns
- **Author statement**: Update immediately with high confidence
