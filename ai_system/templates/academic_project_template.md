# Academic Project Template

## Suggested Directory Structure

### Core Directories
- `research/` - Literature review and research notes
- `sources/` - Citations, references, and bibliography
- `outline/` - Paper structure and organization
- `drafts/` - Writing drafts and versions
- `data/` - Research data, tables, figures (if applicable)
- `notes/` - Working notes and ideas

### Paper Structure

```markdown
# [Paper Title]

## Abstract
[150-250 word summary of the entire paper]

## Introduction
- **Background**: [Context and existing research]
- **Research Question**: [What you're investigating]
- **Significance**: [Why this matters]
- **Thesis Statement**: [Your main argument]
- **Paper Structure**: [Overview of sections]

## Literature Review
- **Theme 1**: [Group of related studies]
  - [Author, Year]: [Key findings]
  - [Author, Year]: [Key findings]
  - **Gap**: [What's missing in current research]

- **Theme 2**: [Another group of related studies]
  - [Author, Year]: [Key findings]
  - [Author, Year]: [Key findings]
  - **Gap**: [What's missing in current research]

## Methodology (if applicable)
- **Research Design**: [Approach used]
- **Data Collection**: [How data was gathered]
- **Analysis Method**: [How data was analyzed]
- **Limitations**: [Constraints and boundaries]

## Analysis/Discussion
- **Finding 1**: [What you discovered]
  - **Evidence**: [Supporting data/quotes]
  - **Interpretation**: [What it means]
  - **Connection to Literature**: [How it relates to existing research]

- **Finding 2**: [What you discovered]
  - **Evidence**: [Supporting data/quotes]
  - **Interpretation**: [What it means]
  - **Connection to Literature**: [How it relates to existing research]

## Conclusion
- **Summary**: [Recap main findings]
- **Implications**: [What this means for the field]
- **Future Research**: [What should be studied next]
- **Final Thoughts**: [Closing statement]

## References
[Full bibliography in required citation style]
```
