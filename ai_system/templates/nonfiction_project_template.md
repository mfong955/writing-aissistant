# Non-Fiction Project Template

## Suggested Directory Structure

### Core Directories
- `outline/` - Book/article structure and organization
- `research/` - Background research and fact-checking
- `sources/` - Citations, references, and bibliography
- `drafts/` - Writing drafts and versions
- `notes/` - Ideas, observations, and working notes
- `interviews/` - Interview transcripts and notes (if applicable)

### Outline Structure

```markdown
# [Project Title] Outline

## Overview
- **Type**: [Book/Essay/Article/Memoir/etc.]
- **Target Length**: [Word count or page count]
- **Target Audience**: [Who will read this]
- **Core Argument/Message**: [Main point or thesis]

## Structure

### Introduction
- **Hook**: [Opening that grabs attention]
- **Context**: [Background information needed]
- **Thesis/Purpose**: [What this work argues or explores]
- **Roadmap**: [What readers can expect]

### Chapter/Section 1: [Title]
- **Main Point**: [Key idea of this section]
- **Supporting Evidence**: [Research, examples, anecdotes]
- **Length**: [Estimated word count]

### Chapter/Section 2: [Title]
- **Main Point**: [Key idea of this section]
- **Supporting Evidence**: [Research, examples, anecdotes]
- **Length**: [Estimated word count]

[Continue for all chapters/sections]

### Conclusion
- **Summary**: [Recap main points]
- **Implications**: [Why this matters]
- **Call to Action**: [What readers should do/think]
```

   - **Relevance**: [How this supports your argument]

2. **[Source Title]** by [Author]
   - **Citation**: [Full citation]
   - **Key Points**: [Main takeaways]
