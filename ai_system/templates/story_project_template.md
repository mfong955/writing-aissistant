# Story Project Template

## Suggested Directory Structure

### Core Directories
- `characters/` - Character profiles and development
- `settings/` - Locations, world-building, environments  
- `plot/` - Story structure, scenes, chapters
- `research/` - Background information, references
- `drafts/` - Writing drafts and versions
- `notes/` - General notes and ideas
