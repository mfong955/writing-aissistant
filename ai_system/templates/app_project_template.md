# Application Project Template

## Suggested Directory Structure

### Core Directories
- `requirements/` - Project requirements and specifications
- `design/` - UI/UX designs, wireframes, mockups
- `architecture/` - System architecture and technical design
- `documentation/` - User guides, API docs, technical docs
- `resources/` - Assets, images, data files
- `planning/` - Project timeline, milestones, tasks
