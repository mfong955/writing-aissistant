## AI Update Instructions

### When to Update This File
- **Session save**: When author says "save session" - update with session accomplishments
- **Milestone completion**: When author completes a significant task or milestone
- **Status changes**: When writing phase changes or blockers are resolved
- **Word count updates**: After each writing session
- **Author request**: When author explicitly asks to update progress

### How to Update
1. **After each session**: Add word count and accomplishments to "Recent Accomplishments"
2. **When sections complete**: Move from "In Progress" to "Completed Sections"
3. **Update word counts**: Adjust "Current Word Count" and calculate progress percentage
4. **Track blockers**: Add new blockers as they arise, move resolved ones to "Resolved Issues"
5. **Update "Next Steps"**: Keep this current based on what author wants to work on next
6. **Log sessions**: Add to "Recent Sessions" with date, word count, and what was accomplished
7. **Celebrate milestones**: Add to "Completed Milestones" when major goals are reached

### What to Update
- **Current Phase**: Change when project moves to new stage (drafting â†’ revising, etc.)
- **Word Count Progress**: Update after each writing session
- **Completed Sections**: Mark chapters/sections as complete with status
- **Recent Accomplishments**: Add session accomplishments (keep last 2 weeks)
- **Active Tasks**: Check off completed, add new ones
- **Blockers**: Add/remove/resolve as status changes
- **Next Steps**: Update based on author's stated priorities
- **Writing Sessions**: Log each session with word count

### What NOT to Do
- Don't mark tasks complete without author confirmation
- Don't remove blockers unless author confirms they're resolved
- Don't change "Next Steps" without author input
- Don't inflate word counts or progress percentages
- Don't assume what the author accomplished - ask or wait for confirmation
